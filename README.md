## 功能列表

- **主要功能（交流QQ群：1040361298）**
  - 自动宝珠迷宫（支持双路线选择：45-50 / 46-38）
  - 角色全装备开孔洗孔（支持多词条筛选）
  - 日常活动副本领活跃一条龙（支持UI可视化编轴）
  - 钓鱼赚取正义币
  - 技术模拟战自动重开

- **辅助功能**
  - UI可视化编轴，墨提斯来了都要说ぜんぶできた
  - 定时启动
  - 电脑关机
  - 启动或结束后打开文件，应用和网站和提醒你自定义文本（比如提醒你微信小助手签到）


## 下载与使用教程
1. **下载可执行文件**  
   - 前往 [Releases页面](https://github.com/nokiruy/Noki-hbr-Auto/releases)
   - 选择最新版本的 `Noki_HBR_Auto-v*.*.*.exe`
   - [视频教程]([https://github.com/nokiruy/Noki-hbr-Auto/releases](https://www.bilibili.com/video/BV1czUvBWEoe/))，用户交流QQ群：1040361298  


本项目基于以下技术实现：
- [OpenCV](https://opencv.org/) 图像识别引擎
- [ADB](https://developer.android.com/tools/adb) 设备控制协议
